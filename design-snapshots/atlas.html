Что хотим

target
layout

Категории — visible zones

Каждая из 5 категорий = тонированный hexagon/cluster с подписью сверху. Внутри — radial-cluster по сложности (центр = базовое, край = advanced). Никаких overlap-ов между зонами.

edges

Стрелки prereq → unlock

Каждое ребро направлено: prereq → ребёнок. Стиль: solid — оба unlocked, dashed — prereq unlocked, ребёнок нет, faded — оба locked.

nodes

Состояние нод считается с одного взгляда

locked = серый outline · unlocked = заливка категорийного цвета · mastered = glow + checkmark · decay = pulse warning ring. Label всегда под нодой, fixed offset, никаких overlap-ов.

interaction

Pan/zoom + клик-и-drawer

Hover на ноду → подсветка предков и потомков (rest faded к 30%). Клик → правый drawer с описанием темы, прогрессом (12 / 18 задач), последним решением, decay-флагом, кнопкой «Решить рекомендованное». Pan/zoom мышью + кнопки +/− и «fit». Mini-map в углу.

Bonus: «twin» режимы

Career mode — раскладка по reading order (от Two Sum к Skip List).
Interview mode — фильтр под конкретную компанию (Яндекс / Ozon / VK), показывает только релевантные ноды и подсвечивает gap.
