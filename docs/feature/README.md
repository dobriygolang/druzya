## Стратегический контекст

**Текущая ниша:** middle/senior разработчики РФ, готовятся к собесу. ~500к человек.

**Расширение должно быть осмысленным**, не «всё для всех». Cluely-trap = смерть. Linear 5 лет был engineering-only — фокус важнее TAM на стадии 0-3к платящих.

### Что добавляем

1. **Senior-треки внутри dev** (System Design depth, Tech Lead, EM prep) — reuse 90% инфры, +30-50% TAM, нулевой урон фокусу.
2. **Sysanalyst + Product analyst** — даёт прирост свитчеров из dev. Требует part-time экспертов (~50к₽/мес/трек). Year 1 H2.
3. **English** — не как Duolingo, а как **operating system between you and your tutor**. Уникальная позиция в РФ-рынке.

### Что НЕ делаем

- ❌ English с нуля / lessons / drills — это Skyeng/Duolingo-территория.
- ❌ Свой language-content — пользователь приносит свой материал.
- ❌ Speech-recognition оценка произношения — бездонная яма.
- ❌ Все 3 трека параллельно. Senior сейчас → English через месяц → Sysanalyst через 4 мес.

### Главный insight по тутору

**Тутор — не платёжеспособный сегмент сам по себе. Тутор — distribution channel.** Приводит платёжеспособных студентов и держит их в экосистеме.

Это превращает druz9 в **двусторонний рынок** с network effect, который защищает от копирования: copy-cat не страшен, потому что нет critical mass пользователей у конкурента.

См [tutor.md](./tutor.md) для деталей.
